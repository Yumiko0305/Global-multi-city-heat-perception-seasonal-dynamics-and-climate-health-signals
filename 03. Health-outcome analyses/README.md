# Code for Health Sub-analysis: Heat Perception vs Temperature Models for Health Outcomes

**Journal:** Nature Communications

---

## Analysis Overview

This repository contains R scripts for the health sub-analysis comparing temperature-based and
heat-perception-based DLNM models across 7 cities in 3 countries:

- **Australia:** Melbourne (emergency department visits)
- **Brazil:** São Paulo, Porto Alegre, Rio de Janeiro (hospitalization)
- **Mexico:** Mexico City, Puerto Vallarta, Playa del Carmen (mortality)

**Outcome types:** Emergency department visits, hospitalization, mortality

**Primary analysis:** Warm season (Mexico: May–September; Brazil/Australia: October–April)

**Sensitivity analysis:** Full-year (all 365 days per city, no seasonal restriction)

## Data Requirements

### Primary input data

**`final_dat.rds`** — City-day level panel dataset. Must be placed in the working directory.

Required columns:

| Column          | Description                                                          |
|-----------------|----------------------------------------------------------------------|
| `city`          | City name                                                            |
| `country`       | Country (Australia, Brazil, Mexico)                                  |
| `date`          | Date (Date class)                                                    |
| `temp`          | Daily mean temperature (°C)                                          |
| `heatPerception`| Heat Perception Intensity Index (HPII)                              |
| `pm25`          | Daily PM2.5 concentration (µg/m³)                                   |
| `holiday`       | Public holiday indicator (0/1)                                       |
| `month`         | Month (integer 1–12)                                                 |
| `dow`           | Day of week (integer)                                                |
| `count`         | Health outcome count (daily)                                         |
| `class`         | Outcome subgroup (all-cause, age0_69, age70_, female, male, ICD codes)|
| `type`          | Outcome type (Hospitlization / Death / Emergency department visit)   |

**Note on data availability:** Health outcome data are available upon request via a data sharing
agreement with the respective data custodians (Melbourne Health, Brazilian Ministry of Health,
Mexican Civil Registry). The perception index (HPII) is derived from social media text data.
Please contact the corresponding author for data access inquiries.

### Intermediate input files (generated by scripts 01–04)

- `which_better/` — xlsx outputs from Script 01 (warm-season model comparison)
- `which_better_allyear/` — xlsx outputs from Script 03 (full-year model comparison)
- `effect_size/` — CSV and PDF outputs from Script 02 (warm-season joint exposure effects)
- `effect_size_allyear/` — CSV and PDF outputs from Script 04 (full-year joint exposure effects)
- `final_v2/data/` — Pre-processed summary CSVs required by Script 05 (figures). These files are derived from the outputs of Scripts 01–04 after manual collation; they are available upon request from the corresponding author.
- `final_v2/data_allyear/` — Pre-processed summary CSVs required by Script 06 (figures). Same provenance as above.

Run scripts in the following order. Set your working directory to the folder containing

## Notes

- Benito Juárez (Cancún area) is excluded from all analyses; it is geographically distinct from
  the other Mexican study sites (>1,000 km from Mexico City).
- The quasi-Poisson family is used throughout; QAIC is computed as
  `-2 * log-likelihood + 2 * df * phi`, where phi is the estimated dispersion parameter.
- Winner thresholds: |ΔQAIC| ≤ 2 → Similar (S); |ΔR²| ≤ 0.02 → Similar (S).
- The crossbasis for Scripts 01 and 03 uses `argvar = list(fun = "lin")` (linear, binary exposure).
- The crossbasis for Scripts 02 and 04 uses `argvar = list(fun = "strata", breaks = c(2,3,4))`
  for the 4-category joint exposure variable.
